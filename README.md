jstool-utils
============

Underscore object '_' with useful functions

``` js
_.isFunction(object)

_.isObject(object)

_.isString(object)

_.isArray(object)

_.isNumber(object)

_.key()

_.keys(object)  // Object.keys() alias

_.extend(object, source)

```
